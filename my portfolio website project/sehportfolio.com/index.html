    Biometric Security System final year Project:

Microcontroller, LCD, KEYPAD,Buzzer, LED, fingerprint module, relay, step down transformer, capacitors, transistors, switches, wires, crystal of 11.9 HZ frequency, EEPROM are the hardware components which was used in my FYP project.
